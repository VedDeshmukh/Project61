My Favorite Coding Languages

If I had the oppurtunity to choose a favorite coding language, I would choose Python and HTML. I would personally choose these two because of the way they work. If you are looking to build a personal portfolio website, your best option is HTML. It is a great language that is easy to use. If I were to pick my favorite coding languages, it would be Python and HTML.
